# Kanban Dashboard

A simple and interactive kanban board application built with vanilla HTML, CSS, and JavaScript. Organize your tasks efficiently across three workflow stages: Todo, In Progress, and Done.

- **[Source Code](your-github-repo-link)** - View the complete code on GitHub
- **[Live Demo](your-live-demo-link)** - Try the application

## Features

- **Three-Column Layout**: Organized workflow with Todo, In Progress, and Done columns
- **Drag & Drop Functionality**: Seamlessly drag tasks between columns
- **Add Tasks**: Quickly add new tasks using the add button
- **Delete Tasks**: Remove tasks with a single click
- **DOM Manipulation**: Built with pure JavaScript using DOM methods and event listeners
- **Responsive Design**: Clean and intuitive user interface

## Technologies Used

- **HTML5**: Semantic structure
- **CSS3**: Styling and layout
- **Vanilla JavaScript**: Interactivity and DOM manipulation

## Usage

1. **Add Task**: Click the "Add" button and enter your task description
2. **Organize**: Drag tasks between columns to update their status
3. **Remove Task**: Click the delete button on any task card to remove it
